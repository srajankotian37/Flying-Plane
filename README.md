# ✈️ Flying Plane

**Flying Plane** is a simple airplane flying game developed using **Unity** and **C#**.
The project demonstrates basic game mechanics such as player movement, environment interaction, and custom-designed assets.

All assets used in the project are **self-designed**, making the project fully original.

---

# 🎮 Game Overview

In **Flying Plane**, the player controls an aircraft and navigates through the environment while maintaining control and avoiding obstacles.
The project focuses on implementing core Unity game development concepts such as physics, object movement, and scene management.

---

# 🛠️ Technologies Used

* **Unity Game Engine**
* **C# (C Sharp)**
* **Unity Physics System**
* **Custom Game Assets**

# ▶️ How to Run the Project

1. Install **Unity Hub**
2. Install a compatible **Unity Editor version**
3. Clone this repository

```bash
git clone https://github.com/srajankotian37/Flying-Plane.git
```

4. Open the project using **Unity Hub**
5. Open the main scene inside the **Scenes** folder
6. Click **Play** in the Unity Editor
